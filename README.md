# Trivia Futbol
Juego de preguntas y respuestas acerca de futbol. El juego consiste en contestar las preguntas en 10 segundos. Si contestas bien ganas el juego.

- Para ver el juego: https://seba0007.github.io/TriviaFutbol/

## Tecnologia Usada
- JQuery
- HTML
- CSS
- Javascript
